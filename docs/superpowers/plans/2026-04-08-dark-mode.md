# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a complete dark mode to Fenua Chords with system preference detection, manual override via toggle, and zero regression in light mode.

**Architecture:** CSS-variable-driven dark mode using the existing `@custom-variant dark (&:is(.dark *))` in Tailwind v4. `next-themes` manages the `.dark` class on `<html>`, persists preference in localStorage, and respects `prefers-color-scheme`. The dark palette ("Lagon de Nuit") redefines all existing CSS variables under a `.dark` selector — no `dark:` Tailwind utilities needed for components already using semantic tokens.

**Tech Stack:** next-themes, Tailwind CSS v4 (class strategy), oklch color space, lucide-react (Sun/Moon icons)

### Task 5: Create ThemeToggle component

**Files:**
- Create: `src/components/ThemeToggle.tsx`

A button that cycles through system → light → dark. Shows Sun icon in dark mode, Moon icon in light mode. Mounted state prevents hydration flash.

- [ ] **Step 1: Create ThemeToggle.tsx**

```tsx
"use client";

import { useEffect, useState } from "react";
import { useTheme } from "next-themes";
import { Sun, Moon, Monitor } from "lucide-react";
import { Button } from "@/components/ui/button";

export function ThemeToggle() {
  const { theme, setTheme } = useTheme();
  const [mounted, setMounted] = useState(false);

  useEffect(() => setMounted(true), []);

  if (!mounted) {
    return <Button variant="ghost" size="icon" className="w-9 h-9" aria-label="Thème" disabled />;
  }

  function cycle() {
    if (theme === "system") setTheme("light");
    else if (theme === "light") setTheme("dark");
    else setTheme("system");
  }

  const icon =
    theme === "dark" ? <Moon className="w-4 h-4" /> :
    theme === "light" ? <Sun className="w-4 h-4" /> :
    <Monitor className="w-4 h-4" />;

  const label =
    theme === "dark" ? "Mode sombre" :
    theme === "light" ? "Mode clair" :
    "Thème système";

  return (
    <Button variant="ghost" size="icon" className="w-9 h-9" onClick={cycle} aria-label={label} title={label}>
      {icon}
    </Button>
  );
}
```
